## 규칙
- 응답은 간결하게. 불필요한 요약/설명 생략
- 코드 탐색은 subagent(Explore)에 위임하여 메인 컨텍스트 보호
- 파일 읽기 시 offset/limit 사용하여 필요한 부분만 읽기
- 큰 작업은 반드시 plan 모드로 시작
- 단계 완료마다 즉시 git commit (WIP 허용)
